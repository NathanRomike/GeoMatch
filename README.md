# GeoMatch - Android

##### 3.18.2016.

#### By **Nathan Romike**

## Description

> This application is to test my knowledge of building android application with Java. Lets make awesome stuff together!

## Setup

*  Clone repository to local machine
*  Start android studio
*  Select GeoMatch application
*  Enter Ctrl+R and select emulator, or connected android device

## Technologies Used

* Java
* XML
* Android Studio
